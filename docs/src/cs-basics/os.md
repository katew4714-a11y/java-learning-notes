# 操作系统

> 操作系统是计算机系统的核心系统软件。

## 核心知识点

### 进程与线程

| 对比 | 进程 | 线程 |
|------|------|------|
| 资源 | 独立资源 | 共享进程资源 |
| 通信 | IPC（管道、消息队列、共享内存） | 直接读写共享数据 |
| 切换 | 开销大 | 开销小 |

### 进程调度算法

- **先来先服务（FCFS）**
- **短作业优先（SJF）**
- **时间片轮转**
- **优先级调度**
- **多级反馈队列**

### 内存管理

- **分页**：固定大小页
- **分段**：逻辑分段
- **虚拟内存**：扩大可用内存

### 死锁条件

1. 互斥
2. 请求与保持
3. 不可剥夺
4. 循环等待
