# 数据结构

> 数据结构是组织和存储数据的方式，是算法的基础。

## 常见数据结构

### 数组

连续内存空间，支持随机访问，插入/删除需移动元素。

### 链表

- **单向链表**：每个节点指向下一个
- **双向链表**：每个节点指向上一个和下一个
- **循环链表**：尾节点指向头节点

### 栈

后进先出（LIFO）。

### 队列

先进先出（FIFO）。

### 树

- **二叉树**：每个节点最多两个子节点
- **二叉搜索树**：左 < 根 < 右
- **平衡二叉树（AVL）**：左右高度差 ≤ 1
- **红黑树**：近似平衡的二叉搜索树

### 哈希表

键值对存储，通过哈希函数映射到数组下标。

## 常用算法

| 算法 | 说明 |
|------|------|
| 排序 | 快速排序、归并排序、堆排序 |
| 查找 | 二分查找、二叉搜索树 |
| 动态规划 | 最优子结构 |
| 贪心 | 局部最优 |
| BFS/DFS | 图遍历 |
