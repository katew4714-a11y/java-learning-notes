# Java 学习指南 🚀

> 一份由浅入深的 Java 学习笔记，记录从基础到进阶的全链路知识体系。

## 为什么写这个笔记

学习 Java 以来，整理了大量的笔记和资料。为了让自己更好地回顾和巩固，也为了帮助更多学习 Java 的小伙伴，我将这些内容整理成了一份知识库。

知识库涵盖 Java 核心、数据库、企业级开发、计算机基础、求职面试等核心知识点。**学 Java，看这个就够了**。

## 知识库地图

知识库分为以下几个模块：

| 模块 | 说明 |
|------|------|
| **Java 基础** | 面向对象、集合框架、IO、异常、Lambda、新特性 |
| **Java 进阶** | 并发编程、JVM 虚拟机 |
| **数据库** | MySQL、Redis |
| **企业级开发** | Spring Boot、MyBatis、Maven、Git、Docker |
| **计算机基础** | 操作系统、计算机网络、数据结构 |
| **求职面试** | 面试题、面经、城市选择 |
| **学习路线** | Java 学习路线、后端学习路线 |
| **项目实战** | 实战项目心得 |
| **程序人生** | 学习方法、职场建议 |

## 如何使用

你可以通过左侧的侧边栏按顺序阅读，也可以根据自己的需求直接跳转到感兴趣的章节。

建议阅读顺序：
1. Java 基础 → Java 进阶 → 数据库 → 企业级开发
2. 穿插计算机基础
3. 准备面试时看求职面试模块
4. 平时翻翻程序人生，调整学习心态
