# 后端学习路线

> Java 后端开发完整学习路线。

## 技能树

```
后端开发
├── 编程语言: Java（主力） + 熟悉 Python/Go
├── 数据库
│   ├── 关系型: MySQL
│   └── NoSQL: Redis、MongoDB
├── 框架
│   ├── Spring Boot / Spring Cloud
│   ├── MyBatis / MyBatis-Plus
│   └── Spring Security / Shiro
├── 中间件
│   ├── 消息队列: Kafka、RabbitMQ
│   ├── 搜索引擎: Elasticsearch
│   └── 注册中心: Nacos、Eureka
├── 工具
│   ├── Maven / Gradle
│   ├── Git
│   └── Docker / K8s
├── 计算机基础
│   ├── 操作系统
│   ├── 计算机网络
│   └── 数据结构与算法
└── 软技能
    ├── 文档写作
    └── 沟通协作
```

## 学习顺序建议

1. Java 基础 → MySQL → Spring Boot → MyBatis
2. 并发编程 → JVM → Redis
3. Spring Cloud → 消息队列 → Docker
4. 计算机基础（贯穿学习）
5. 项目实战（以上每阶段都可穿插）
