# Java 学习路线

> 一份完整的 Java 学习路线，从零基础到可就业。

## 第一阶段：Java 基础（1~2 个月）

| 知识点 | 要求 |
|--------|------|
| Java 语法 | 变量、运算符、流程控制 |
| 面向对象 | 类、对象、继承、多态、接口 |
| 集合框架 | List、Set、Map 的常见实现 |
| IO 流 | 字节流、字符流、缓冲流 |
| 异常机制 | try-catch、自定义异常 |
| 多线程 | Thread、Runnable、synchronized |

**推荐资源**：
- 《Java 核心技术 卷I》
- 尚硅谷/黑马 Java 基础视频

## 第二阶段：数据库（2~4 周）

- MySQL：SQL 语法、索引、事务、JDBC
- Redis：五大数据结构、持久化

## 第三阶段：Java Web（4~6 周）

- HTML/CSS/JavaScript（了解即可）
- Servlet、Tomcat
- Spring Boot、MyBatis

## 第四阶段：进阶（持续学习）

- 并发编程：JUC、线程池、锁
- JVM：内存模型、GC 调优
- Spring 源码
- 分布式：微服务、消息队列

## 第五阶段：项目实战

### 项目选择

| 难度 | 项目 |
|:----:|------|
| ⭐ | 个人博客、管理系统 |
| ⭐⭐ | 秒杀系统、论坛 |
| ⭐⭐⭐ | 微服务项目、RPC 框架 |
