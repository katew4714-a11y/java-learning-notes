# 学习方法

> 高效的学习方法能让你的成长事半功倍。

## 费曼学习法

**核心**：通过教别人来检验自己的理解。

1. 选择概念
2. 用自己的话解释给不懂的人听
3. 发现解释不清的地方就回去复习
4. 简化表达

## 刻意练习

**不是盲目重复**，而是有目的的、有针对性的练习：

- 明确目标
- 走出舒适区
- 获得反馈
- 反复改进

## 学习建议

### 多阅读源码

阅读优质开源项目源码是提升最快的方式之一。

### 多写笔记

好记性不如烂笔头。写笔记的过程就是二次学习。

### 勤实践

光看不练是学不会编程的。每个知识点学完后立刻动手写代码验证。

### 坚持

学习编程是一场马拉松，不是百米冲刺。每天进步一点点，长期下来就是巨大的差距。
