# 职场建议

> 给程序员的一些职场成长建议。

## 技术成长

### 深耕一门语言

先把一门语言学到精通，再横向扩展。

### 关注底层原理

不要只会用框架，要理解框架背后的原理。

### 保持学习

技术更新迭代很快，保持持续学习的习惯。

## 软技能

### 沟通能力

- 技术方案的表达要清晰
- 及时同步进度
- 提出问题时带上自己的思考

### 文档能力

好的文档能让你和团队都受益。

### 时间管理

- 学会区分优先级
- 专注重要的事情
- 避免多任务并行

## 职业规划

### 1~3 年

打好基础，快速积累技术经验。

### 3~5 年

成为某个领域的专家，能独立负责模块。

### 5~8 年

技术架构能力 + 团队管理能力。

### 8 年+

要么深耕技术（架构师），要么转管理（技术经理/CTO）。
