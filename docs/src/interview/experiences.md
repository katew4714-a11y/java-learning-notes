# 优质面经

> 真实面试经验分享，帮助你更好地准备面试。

## 如何准备面试

1. **基础知识**：Java 基础、集合、并发、JVM
2. **框架**：Spring Boot、MyBatis
3. **数据库**：MySQL、Redis
4. **项目**：梳理项目亮点和难点
5. **算法**：LeetCode 高频题
6. **系统设计**：高可用、高并发、分布式

## 面试流程

- **一面（技术面）**：基础知识 + 项目
- **二面（技术面）**：深入原理 + 系统设计
- **三面（主管面）**：业务理解 + 技术广度
- **HR 面**：沟通能力 + 职业规划

## 常见问题

**Q：项目经验怎么写？**

A: 
- STAR 法则：Situation、Task、Action、Result
- 突出技术难点和解决方案
- 量化结果（性能提升多少、并发量等）

**Q：不会的问题怎么办？**

A: 不要慌张，坦诚说明不了解，但可以尝试分析思路。面试官更看重学习能力和思维方式。
