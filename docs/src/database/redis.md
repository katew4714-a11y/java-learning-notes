# Redis

> Redis 是一个高性能的键值存储系统，常用于缓存、分布式锁等场景。

## 数据结构

| 类型 | 说明 | 使用场景 |
|------|------|----------|
| String | 字符串 | 缓存、计数器 |
| Hash | 哈希 | 对象存储 |
| List | 列表 | 消息队列 |
| Set | 无序集合 | 标签、去重 |
| ZSet | 有序集合 | 排行榜 |
| Bitmap | 位图 | 签到统计 |
| HyperLogLog | 基数统计 | UV 统计 |
| GEO | 地理位置 | 附近的人 |

## 持久化

| 方式 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| RDB | 快照 | 文件小，恢复快 | 可能丢数据 |
| AOF | 追加日志 | 数据完整 | 文件大，恢复慢 |

## 高可用

- **主从复制**：读写分离
- **哨兵模式**：自动故障转移
- **Redis Cluster**：分布式集群

## 常见问题

### 缓存穿透

查询一个不存在的数据，绕过缓存直接查数据库。

**解决**：布隆过滤器、缓存空值

### 缓存击穿

热点 key 过期，大量请求打到数据库。

**解决**：互斥锁、热点数据不过期

### 缓存雪崩

大量 key 同时过期，DB 压力暴增。

**解决**：过期时间加随机值、多级缓存
